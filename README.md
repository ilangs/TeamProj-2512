# 4 Team Project: LLM 스크립트 생성기
이 프로젝트는 LLM 프롬프트를 활용한 광고카피,유튜브,SNS 멀티 스크립트 생성 도구입니다.

## 작업 이력
- **main**: 최신 안정화 버전 (22일, 23일 작업분)
   * 12/18 유튜브 스크립트 생성기 구현 (CLI, Streamlit)
   * 12/19 프롬프트 탬플릿 모듈 분리
   * 12/22 광고 카피/SNS Streamlit 구현 및 ask-llm/moderation 모듈 분리
   * 12/22 tts 모듈 추가
   * 12/23 가독성 위한 개별 주석 추가
- **trial-success**: 개별시도 기록 보관
- **trial-fail**: 개별시도 실패 기록 보관 (차후 재시도)

## 주요 기능
- 광고카피, 유튜브, SNS 멀티 스크립트 생성
- Streamlit 기반 웹 UI 인터페이스
- 프롬프트 템플릿 최적화
- Best 스크립트 음성 변환
